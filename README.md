# Stylus_-Font-rendering  
使用方法：  
浏览器安装Stylus 插件，将Font rendering.css内容复制，然后再Stylus中新建css样式，将复制的内容粘贴即可。  

注释：  
"ADaQianZiBi"为字体名称，修改对应的字体修改名称即可。  
作者：百度 Mactype贴吧"多兰的妖术"
